# Structural Repo Context

Created: 2026-04-13T22:00:56Z

## Summary

Add a lightweight structural repo layer so Brain can orient an agent around boundaries, entrypoints, tests, and configuration without pretending to have full semantic code intelligence.

## Why It Matters

Brain has good durable markdown context but no always-on compressed repo understanding layer. A structural map should improve orientation and context selection before deeper retrieval is needed.
