# V1 Context Packet Compiler

Created: 2026-04-15T03:40:00Z

## Summary

Add the first deterministic `brain context compile` surface that assembles a compact working-set packet from base contract items, changed files, boundaries, nearby tests, and durable note summaries.

## Why It Matters

This is the visible product shift. Once Brain can compile a justified task packet instead of mostly loading documents, the product starts to answer its new core question directly: what should the agent know right now?
