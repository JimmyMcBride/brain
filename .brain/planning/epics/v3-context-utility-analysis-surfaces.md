# V3 Context Utility Analysis Surfaces

Created: 2026-04-15T03:40:00Z

## Summary

Add human-facing inspection surfaces that show why context was included, what got expanded, and which items appear to be signal or noise over time.

## Why It Matters

Telemetry without inspection turns the compiler into a black box. This epic keeps Brain explainable by exposing packet rationale and utility analysis in commands people can actually inspect and debug.
