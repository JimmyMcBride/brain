# V4 Context Compiler UX Migration

Created: 2026-04-15T03:40:00Z

## Summary

Finish the product shift by making `context compile` the primary mental model, shrinking the old static-level framing, and updating docs, skills, and command guidance to match the compiler architecture.

## Why It Matters

Even if the internals change, Brain will still feel document-centric if the UX and docs keep teaching the old model. This epic makes the new compiler architecture the actual product story.
