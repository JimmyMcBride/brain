# Add Conservative Utility Wording And Diagnostic Fields

Created: 2026-04-15T05:34:09Z

## Description

Keep utility inspection trustworthy by introducing conservative language and explicit diagnostic fields that explain what Brain observed without overstating weak evidence.


## Acceptance Criteria

- [ ] Explain and stats surfaces use conservative wording for likely utility and likely noise judgments
- [ ] Diagnostic fields such as include counts, expansion counts, and successful verification linkage are visible where they support user trust
- [ ] The wording avoids implying that weak telemetry is definitive truth about context quality
