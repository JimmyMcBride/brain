# Add Local Packet Telemetry Event Model And Storage

Created: 2026-04-15T05:34:09Z

## Description

Add the local storage model and event families needed to record compiler packet activity without drifting into generic analytics or transcript capture.


## Acceptance Criteria

- [ ] Brain has a local packet-telemetry model for packet compilation, expansion, verification, durable-update, and closeout events
- [ ] Telemetry storage stays local, inspectable, migration-friendly, and bounded in scope to compiler usefulness
- [ ] The initial event model records factual compiler usage without storing chain-of-thought or broad scratch state
