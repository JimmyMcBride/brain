# Blend Utility With Deterministic Task Boundary And Workflow Signals

Created: 2026-04-15T05:34:09Z

## Description

Blend utility signals into the existing selector so Brain still respects task match, boundary overlap, and workflow-required items while getting modestly smarter from repeated local outcomes.


## Acceptance Criteria

- [ ] Utility-aware ranking is blended with deterministic task, boundary, changed-file, and workflow-required signals instead of replacing them
- [ ] Ranking remains deterministic for the same repo state and telemetry state
- [ ] Obvious task or boundary mismatches cannot be masked by a strong utility history alone
