# Define Promotable And Non-Promotable Categories For Compiler-Era Brain

Created: 2026-04-15T05:34:50Z

## Description

Define the first-wave category system that says which compiler-era findings are eligible for durable promotion and which kinds of ephemeral session state should die by default.


## Acceptance Criteria

- [ ] Brain has explicit first-wave promotable categories such as decisions, invariants, gotchas, verification recipes, boundary facts, and follow-ups
- [ ] Brain has explicit non-promotable defaults for speculative reasoning, transient scratch, and dead-end experiments unless they are reviewed as recurring traps
- [ ] The category model is clear enough to drive later promotion gating and closeout suggestions without implying transcript capture
