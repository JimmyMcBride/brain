# Integrate Verification And Test Surfaces Into Compiled Packets

Created: 2026-04-15T05:33:26Z

## Description

Integrate nearby tests and verification recipes into packet output so implementation-oriented packets answer both what context matters and how the result should be checked.


## Acceptance Criteria

- [ ] Compiled packets can include the strongest nearby tests and verification recipes with explicit reasons and bounded output size
- [ ] The compiler distinguishes between strongly recommended verification and weaker suggested checks
- [ ] Verification and test surfaces improve packet usefulness without hiding their provenance or crowding out the main working set
