Created: 2026-04-11T14:49:34Z

## Description

Keep sessions as a strong feature, but make sure the default enforcement profile matches current product maturity and solo-developer reality. Review which rules should be default and which should be opt-in via policy.


## Acceptance Criteria

- [ ] Session defaults feel lightweight enough for everyday use
- [ ] Stricter enforcement remains available through policy
- [ ] Docs present sessions as a tool instead of mandatory ceremony
