# Tune Closeout Suggestion UX And Suppression Rules

Created: 2026-04-15T05:34:50Z

## Description

Tune the closeout UX so promotion suggestions feel useful and bounded, including suppression rules for duplicate or low-value suggestions across repeated sessions.


## Acceptance Criteria

- [ ] Closeout suggestions use concise, review-first wording rather than nagging or transcript-like prompts
- [ ] Suppression rules prevent repeated duplicate suggestions across similar sessions when nothing materially changed
- [ ] Tests or fixtures cover the silent-closeout path as well as the bounded-suggestion path
